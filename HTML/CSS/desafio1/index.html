Estrutura Básica

(Voltar)

Um documento HTML precisa seguir obrigatoriamente a estrutura compsta pelas tags <html>, <head> e <body> e a instrução <!DOCTYPE>.


Entendendo as Tags

(Voltar)

A tag <html> é a tag "mãe" e dentro dela ficam as tags <head> e <body> são as tags "filhas".

A tag <head> é onde contém as informações sobre o documento HTML, que interessam somente ao navegador e serviços da web. Ninguém vê o que esta na tag <head>.

Enquanto na tag <body> é onde fica todo o corpo do Documento HTML e que é exibido pelo navegador.
